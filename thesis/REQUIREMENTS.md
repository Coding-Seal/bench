# Требования к ВКР — выжимка

Источник: `thesis/sources/requirements.pdf` (46 с.) — «Содержание, структура и
оформление ВКР», СПбПУ/ВШПИ, направление 09.03.04, одобрено 01.04.2025.
Базируется на ГОСТ 7.32-2017. Этот файл — дистиллят, чтобы не перечитывать PDF.
При сомнении сверяться с исходным пунктом (в скобках указан номер).

## Стиль изложения — п. 8.1 (КРИТИЧНО, частый источник нарушений)

- Строгий научно-литературный стиль. **Запрещены** свободные формы
  художественной беллетристики, дружеской переписки, разговорные обороты.
  (Маркеры-нарушители: «дрожит», «трогает», «гулял», «наконец-то отрабатывает»,
  «Получился прототип», «Главный вывод простой», «дотюнивать на лету» и т. п.)
- Предложения чёткие и однозначные: всегда ясно — сделано действие или нет,
  получен результат или нет, авторское это или заимствованное.
- Завершённое прошедшее время, описание подтверждённых результатов.
- Логическая связность: каждая часть вытекает из предыдущей / из цели и задач.
- Вместо иностранных терминов — русские аналоги; если аналога нет, термин
  выносится в раздел «Определения, обозначения и сокращения».
  ⚠ Открытый вопрос в текущей работе: «сэмплер», «бенчмарк», «скрапинг»
  используются по всему тексту — либо заменить, либо внести в перечень.

Цель «не выглядеть как ИИ» достигается вариативностью структуры и снятием
ИИ-маркеров (жирные псевдозаголовки с `:`, механические триплеты,
«это типичное поведение…»), а НЕ переходом в разговорный регистр.

## Структура ВКР — п. 7.1

Титульный лист → Задание → Реферат (рус+англ, разные страницы) → Содержание →
Определения/обозначения/сокращения (по необходимости) → Введение →
Основная часть → Заключение → Список использованных источников →
Приложения (при наличии).

- Реферат: 1000–1500 печатных знаков; ключевые слова прописными, в строку,
  через запятые, без точки в конце (п. 7.4, 8.4).
- Введение, заключение, список источников, приложения — **не нумеруются**
  (п. 7.5, 8.8). Их `=`-заголовки — структурные элементы (шаблон это знает).
- Раздел «Определения…» добавляется, если ≥ 5 необщепринятых аббревиатур;
  тогда он становится первым разделом с нумерацией страниц (п. 8.6).
- Перечень сокращений: слева без абзацного отступа в алфавитном порядке,
  справа через тире расшифровка, столбцом, без знаков препинания в конце
  строки (п. 8.7).

## Относительные объёмы (п. 7.7–7.9)

| Часть | Доля |
|---|---|
| Введение | ~5 % |
| Анализ предметной области (гл. 1) | ~35 % |
| Теоретическое предложение / новизна (гл. 2) | ~20 % |
| Технологическое решение: архитектура, реализация (гл. 3) | ~20 % |
| Апробация и анализ результатов (гл. 4) | ~15 % |
| Заключение | ~5 % |

Объём бакалаврской без титула/содержания/списка/приложений: **40–60 страниц**
(магистратура 60–100). Структура содержательной части (гл. 1→4) — п. 7.8.

## Содержание глав (п. 7.8)

- Введение: актуальность, состояние проблемы, значимость, степень
  разработанности, цель, задачи, структура и объём. Без схем/таблиц/рисунков.
  Задач — **не менее 4 и не более 6**, формулировки глаголами
  («изучить», «проанализировать», «разработать», «описать»).
- Гл. 1 «Анализ предметной области»: критический анализ теоретических и
  технологических решений, набор критериев сравнения, итоговая
  сравнительная таблица с выявлением недостатков, выводы как требования к
  результату. Без «пустого» цитирования.
- Гл. 2 «Теоретическое предложение»: обоснование теоретического базиса,
  авторское предложение (новизна) на математическом языке, выводы.
- Гл. 3 «Технологическое решение»: концептуальная схема, архитектура с
  чётким разделением авторских и заимствованных частей, обоснование выбора
  алгоритмов/библиотек/среды. Исходный код — только в приложение.
- Гл. 4 «Апробация и анализ результатов»: система метрик (применена и к
  своему решению, и к существующим из гл. 1), сравнительный анализ,
  графики/диаграммы, акцент на достоверности статистики, факт достижения цели.
- Заключение: последовательное обобщение выводов по разделам; формальное
  перечисление решённых задач (совпадает со списком из Введения); без новой
  информации.

## Оформление текста (п. 8.2–8.9)

- A4, Times New Roman 14, чёрный, межстрочный 1.5 (18 pt), абзац 1.25 см.
- Поля: внутреннее 30, внешнее 15, верхнее/нижнее 20 мм.
- Выравнивание по ширине, автоперенос; в заголовках переносы запрещены.
- Нумерация страниц — посередине, внизу, 10 мм от края, арабские, без точки;
  отсчёт с титула, отображение с «ВВЕДЕНИЕ» (или «ОПРЕДЕЛЕНИЯ…»).
- Структурные заголовки: по центру, ПРОПИСНЫМИ, без точки, не подчёркивать,
  каждый — с новой страницы.
- Главы — арабские с точкой (`2.1.3.`); заголовки уровней оформляются
  по-разному, выделяются интервалом 6 pt.
- Всё перечисленное реализует шаблон `modern-g7-32` — вручную не настраивать.

## Рисунки / таблицы / формулы / списки (п. 8.10–8.16)

- Рисунок: по центру, после ссылки в тексте. Подпись под рисунком, по центру,
  с прописной, без точки. **Формат методички: «Рисунок N. Название».**
  ⚠ Шаблон выводит «Рисунок N — Название» (стиль ГОСТ 7.32). Расхождение
  существует; правка — только на уровне шаблона, не текста.
- Таблица: после ссылки. Наименование над таблицей слева без абзацного
  отступа: «Таблица N. Название», без точки. Перенос: «Продолжение таблицы».
  Заголовки граф — с прописной, по центру; единицы измерения обязательны.
- Формулы: отдельной строкой, по центру, нумерация в круглых скобках справа
  сквозная. Пояснение символов — сразу под формулой, первая строка со слова
  «где» без двоеточия, с абзаца, каждый символ с новой строки.
- Ссылки в тексте: «в соответствии с рисунком 2», «приведены в таблице 1»,
  «в формуле (1)», «данные приводятся в приложении B». На всё должны быть
  ссылки (п. 8.14).
- Источники: `[n]` арабскими в квадратных скобках, сквозная нумерация по
  порядку появления; в списке нумерация арабскими с точкой, с абзаца.
  Формы: `[2, с. 123-124]`, `[2, раздел 5]`, `[2]` (п. 8.15–8.16).
- Списки: маркированный — тире; нумерованный — `1)` (или кириллица
  `enum-numbering`, исключая ё,з,й,о,ч,ъ,ы,ь). Пункты через `;`,
  последний — `.` (п. 8.6 doc шаблона / ГОСТ 6.4.6).

## Приложения (п. 8.17)

После списка источников; каждое с новой страницы; «Приложение» вверху по
центру; буквы кириллицы с А (искл. Ё,З,Й,О,Ч,Ъ,Ы,Ь) либо латиницы (искл. I,O);
заголовок приложения — с прописной, полужирный, по центру, без точки;
все приложения перечислены в содержании.
